# MediaPipe 视觉处理工作台 (`README.md`)

基于 **Python**、**PySide6** 与 **MediaPipe** 开发的轻量级桌面端视觉处理工作台。支持实时调用本地摄像头进行多种主流计算机视觉与机器学习任务。

## 功能特性

- **手势识别 (Hands)**：实时检测双手并绘制手部关键点及骨骼连接。
- **全身骨骼 (Pose)**：实时捕捉人体全身骨骼姿态并进行连线渲染。
- **背景分割 (Seg)**：基于人像分割技术，实时实现人景分离与背景替换。
- **3D 物体 (Objectron)**：支持对特定 3D 物体（如鞋子等）进行边界框与姿态轴估计。
- **头发分割 (Hair)**：利用自定义 `.tflite` 模型精准实现头发区域高亮与蒙版叠加。
- **瞳孔追踪 (Mesh)**：精准捕捉双眼瞳孔位置并进行红点标记。
- **高精度轮廓 (Full)**：精简渲染面部关键轮廓线条（基于 `FACEMESH_CONTOURS`），清晰勾勒脸部轮廓而不显杂乱。
- **人脸检测 (Det)**：快速框出画面中的所有人脸。

## 运行环境

- **Python 版本**：推荐 Python 3.8 ~ 3.10
- **依赖库**：见上方 `requirements.txt`

## 快速上手

1. 安装依赖库：

   Bash

   ```
   pip install -r requirements.txt
   ```

2. *(可选)* 如果需要运行**头发分割**功能，请确保修改代码中的 `model_path` 指向您本地的 `hair_segmenter.tflite` 模型文件。

3. 运行主程序：

   Bash

   ```
   python main.py
   ```
